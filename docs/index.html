Cyrillic ↔ Latin Transliterator

Convert text between Latin and Cyrillic alphabets instantly. This tool supports multiple Slavic languages and internationally recognized transliteration standards.

Supported Languages & Standards

  • Serbian – Azbuka
  • Russian – ISO 9:1995
  • Ukrainian – ISO 9:1995
  • Bulgarian – ISO 9:1995
  • Belarusian – ISO 9:1995
  • Macedonian – ISO 9:1995
  • Macedonian – ALA-LC
