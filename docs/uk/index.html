Транслітератор Кирилиця ↔ Латиниця

Миттєво перетворюйте текст між латиницею та кирилицею. Інструмент підтримує кілька слов’янських мов і міжнародні стандарти транслітерації.

Підтримувані мови та стандарти

  • Сербська — Азбука
  • Російська — ISO 9:1995
  • Українська — ISO 9:1995
  • Болгарська — ISO 9:1995
  • Білоруська — ISO 9:1995
  • Македонська — ISO 9:1995
  • Македонська — ALA-LC
